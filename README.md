# 在庫管理アプリ

## 📌 概要
シンプルな在庫管理ができるWebアプリです。  
商品の登録・更新・削除に加え、削除データの復元機能も実装しています。

---

## 🛠 機能一覧

- 商品の登録
- 在庫数の追加・更新
- 在庫の削除（論理削除）
- 削除済み一覧の表示
- 削除データの復元

---

## 💡 工夫した点

### 論理削除を採用
通常の削除（DELETE）ではなく、`is_deleted` フラグで管理することで、  
データの復元が可能な設計にしました。

### 削除済みデータの管理画面
削除されたデータを確認できる専用ページを作成し、  
復元操作ができるようにしました。

---

## 🧠 データ構造

### 概要
本アプリでは、在庫を辞書型（dictionary）およびデータベースで管理しています。

### 構造
在庫は「商品名」と「数量」のペアで構成されます。

- キー：商品名
- 値：数量

### 採用理由
- 商品名から数量をすぐに取得できるため
- シンプルで理解しやすいため
- 現在の要件（商品名と数量のみ）に適しているため

### データ例
```json
{
  "コーヒー（コロンビア）": 3,
  "いちご": 6,
  "生クリーム": 3
}
